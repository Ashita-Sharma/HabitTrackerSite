## Features
- User signup & login
- Password hashing
- Session-based authentication
- Delete users (admin feature)
- Access Database (debug feature)
- Handling Task priority
- Sorting Tasks by Priority or Deadline
- Handling Task Edits and modification
- Added Search Query for tasks
- Added handling cases where a user attempts to login or sign up during an active session
- Handling a case where an email is already in use.
- Added ability to edit display name and email
- Log Out feature

## Tech Stack
- Flask
- SQLite
- HTML / Jinja2

## How to Run
1. pip install flask
2. python app.py
3. open http://127.0.0.1:5000
