## Features
- User signup & login
- Password hashing
- Session-based authentication
- Delete users (admin feature)

## Tech Stack
- Flask
- SQLite
- HTML / Jinja2

## How to Run
1. pip install flask
2. python app.py
3. open http://127.0.0.1:5000
